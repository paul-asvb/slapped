# Game Design Concept - Top-Down Combat Racing Game
# Optimized for AI Sprite Generation (Ludi.ai)

## Art Style Reference
Primary Inspiration: "Wrecked: Revenge Revisited" (2012)
- Colorful, vibrant arcade aesthetic
- Stylized cartoon look with bold outlines
- Exaggerated proportions for readability at small sizes
- High contrast colors for gameplay clarity
- Glossy, toy-like vehicle designs

## Visual Style Keywords
- Cartoon racing
- Isometric/top-down perspective
- Vibrant saturated colors
- Cel-shaded look
- Arcade style
- Toy-like vehicles
- Bold black outlines
- Clean vector style

---

## VEHICLES (Top-Down View)

### Style Guidelines
- Viewed from directly above (bird's eye view)
- Chunky, rounded shapes
- Visible wheels on sides
- Exaggerated features (big exhausts, spoilers)
- Clear front/back orientation
- Size: ~64x64 or 128x128 pixels

## GENERAL PROMPT TIPS FOR LUDI.AI

1. Always specify "top-down view" or "bird's eye view"
2. Include "game sprite" or "arcade game asset"
3. Add "white background" for easy extraction
4. Use "black outline" for cartoon style
5. Mention "clean", "simple", "bold" for clarity
6. Specify pixel size if possible (64x64, 128x128)
7. Add "transparent background" for effects/particles

## Example Full Prompt Structure
"[View angle], [art style] [object], [color/material], [key features], [style keywords], [outline style], [purpose], [background]"
